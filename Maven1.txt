Clean the files and directories generated by Maven during its build.
Clean the target directory
 mvn clean:clean
Mostly used with other mvn commands

mvn test
run tests using a suitable unit testing framework:
Default
run the test cases of the project using the maven-surefire-plugin
Execute all tests from classes which ends with “Test”
Execute all test cases tagged with @Test
# Run a single test class.
$ mvn -Dtest=TestApp1 test

# Run multiple test classes.
$ mvn -Dtest=TestApp1,TestApp2 test

# Run a single test method from a test class.
$ mvn -Dtest=TestApp1#methodname test

# Run all test methods that match pattern 'testHello*' from a test class.
$ mvn -Dtest=TestApp1#testHello* test

# Run all test methods match pattern 'testHello*' and 'testMagic*' from a test class.
$ mvn -Dtest=TestApp1#testHello*+testMagic* test
